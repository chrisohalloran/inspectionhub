# Building inspection drafting

Load this skill only for a commissioned Building module in a frozen investigation packet.

- Describe the visually observed condition, location, apparent extent, uncertainty, relevant consequence, technical further investigation and inspector-stated conclusion.
- Preserve distinctions between observed fact, inspector opinion, assumption and possibility.
- Repeat a major-defect, minor-defect or safety-hazard classification only when an inspector-authored packet source states it. Attribute the classification to the inspector.
- Treat concealed construction and mechanism statements as possibilities unless directly observed.
- Cite one or more exact packet sources for every factual clause.
- Keep limitations visible and specific to the area and access available at inspection time.
- Do not determine compliance, prescribe repair work, estimate cost or give purchase, negotiation, legal, settlement, warranty or timing advice.
- Do not introduce Timber Pest categories into the Building module.

This skill does not reproduce AS 4349.1 and does not claim that the draft requirement matrix has received licensed professional sign-off.
